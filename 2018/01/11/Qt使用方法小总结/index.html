Qt使用方法小总结 
1.qpushbutton

Qpushbutton是qt中的按钮，可以对其进行resize,move,settext,setfont等操作，setfont的参数是qfont类型qfont第一个参数是qstring类型，一般用“”代替就行，第二个参数是字号的大小。

3.qlabel

4.qtableview

temp1=new QStandardItemModel(this);//建立qstanditemmodel
ui->tableView->setModel(temp1);//将tableview与standitemmodel结合
ui->tableView->setEditTriggers(QAbstractItemView::NoEditTriggers);//不能编辑
ui->tableView->setSelectionBehavior(QAbstractItemView::SelectRows);//选中时，选中行
ui->tableView->setSortingEnabled(true);//可以排序
ui->tableView->setShowGrid(false);//不显示网格
ui->tableView->verticalHeader()->hide();//隐藏头一列

temp1->setHorizontalHeaderItem(0,new QStandardItem(tr(“课程名称”)));
temp1->setHorizontalHeaderItem(1,new QStandardItem(tr(“上课时间”)));
temp1->setHorizontalHeaderItem(2,new QStandardItem(tr(“教室”)));

temp1->setItem(index,j++,new QStandardItem(te.split(“ “)[0]));
temp1->setItem(index,j++,new QStandardItem(te.split(“ “)[1]));

获取表格的数据，要用temp而不是tableview
temp1->data(temp1->index(ui->tableView->currentIndex().row(),1)).toString()
data的参数是index对象，index包括row和column一般可以用temp.index来表示

5.qfile

QFile p(“路径”);
p.open(QIODevice::ReadOnly);//只读，writeonly和readwrite等多种模式。
QTextStream pp(&p);//绑定
pp.setCodec(“System”);//设置编码
QString te;
while (!pp.atEnd())//不到文件最后
{
te=pp.readLine();//pp.readline() p.readline()不同点是第一个最后没有“\r\n”的回车换行，但第二个有。
}
文件写入的话就用<<，将指针放到开头p.resize(0);

7.QDebug

qDebug()<< 输出信息供debug使用

8.qfont 字体设置，第一个参数是qstring对象，一般用空字符串代替，第二个是字号大小

9.qmessagebox

QMessageBox::warning(this,tr(“通知”),ii.readAll(),tr(“确定”));//第一个参数父窗口，第二个窗口名称，第三个窗口内容，第四个是按钮名称。
int reply=QMessageBox::question(this,tr(“提示”),tr(“确定要删除这一节课吗”));//第一个是父窗口，第二个是名称第三个是显示内容，返回值一般有两种QMessage::yes,QMessange::no

10.qbrush,qpixmap qpalette设置背景图片

QPalette temp;//调色板
temp.setBrush(QPalette::Background,QBrush(QPixmap(“./doc/image/3.jpg”)));//qpix将图片像素点提取出来，通过qbrush刷到调色板上
this->setPalette(temp);//给窗口设置调色板。

11.设置透明度和颜色

log->setStyleSheet(“background:rgba(250,250,250,100);”);

12.槽和信号的连接

connect(log,&QPushButton::clicked,this,&student_log::login);
connect(exit,&QPushButton::clicked,this,&student_log::ex);

13.窗口设置

this->setWindowFlags(Qt::FramelessWindowHint);//窗口无标题栏
this->setWindowOpacity(0.2);//窗口透明度
